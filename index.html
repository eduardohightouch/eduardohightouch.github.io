Event Collection Demo

This page showcases the Hightouch Browser (JavaScript) SDK. Every button below fires a real window.htevents.track() call, and every call is echoed live in the Event Log panel so you can see the SDK working without opening your network tab.

See the Browser SDK docs for the full API reference.

Before events will actually reach Hightouch, open index.html and replace YOUR_WRITE_KEY in the snippet with the write key from your Event Source. Until then, the SDK will still load and the demo will still work locally — the log panel below reflects what your code is calling, regardless of whether the write key is valid.

Track events

Each button calls htevents.track(event, properties, context, callback) with a distinct event name and a properties object.

Identify user

Calls htevents.identify(userId, traits) to associate future events with a known user.

Event log

Every SDK call made on this page appears here as it happens.
