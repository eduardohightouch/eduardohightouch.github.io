Campaign Attribution / UTM Demo

Hightouch automatically captures UTM parameters that you pass in URLs. You can also manually pass UTMs into the context.campaign field.

When a visitor arrives via a link containing UTM parameters, the Browser SDK auto-parses them into context.campaign on the automatic htevents.page() call that fires when this page loads — no manual work required.

Manually pass UTMs

You can also set context.campaign explicitly on any page() or track() call, matching the documented example:

Auto-captured UTM params from the URL do not persist to subsequent calls unless you set them explicitly in context.campaign — each call only carries the campaign data you pass to it (or, for the initial automatic page() call, what was parsed from the URL at load time).
